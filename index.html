Regression Pipeline Analysis Report

California Housing Valuation Model

1. Model Objective & Background

This project implements an Ordinary Least Squares (OLS) Linear Regression estimator targeting median house prices in California block groups. By analyzing demographic metrics (household sizes, rooms, geographic slopes), the estimator establishes weights coordinates explaining regional valuations.

2. Optimization Fitting Metrics

Optimization Parameter Model Benchmark Metric Value
Mean Absolute Error (MAE) $53,320.01
Root Mean Sq. Error (RMSE) $74,558.14
Coefficient of Determination (R²) 0.5758 (57.58%)

3. Features Influencing Slopes

  • Median Income: Strongest positive weight coefficient. Affluent areas show significant price gains.
  • House Age: Gentle positive trend. Established blocks carrying historic or premium value scores.
  • Average Occupants: Negative correlation weight. Overcrowded blocks exhibit reduced value indexes.
